CLI#

For each command the gray box shows the usage.

These are documented according to bash/unix convention.

  • the [] do not get used, they indicate that there might be more than one item in that position

  • [OPTIONS] refers to any optional inputs or options

  • [ARGS] refers to required inputs, or arguments

cspt#

cspt [OPTIONS] COMMAND [ARGS]...

badge#

log an additional badge that is not in the pr list

cspt badge [OPTIONS] BADGE_TITLE

Options

-i, --info <info>#

messgae to include for comment

-v, --to-terminal#

write to terminal instead of a file

Arguments

BADGE_TITLE#

Required argument

badgecounts#

check if early bonus is met from output of gh pr list -s all –json title,latestReviews,createdAt and return a message. input from either a file or -for stdin

cspt badgecounts [OPTIONS] JSON_OUTPUT

Options

-s, --soft#

soft check, skip title check

-m, --manual#

check for manually recorded badges

Arguments

JSON_OUTPUT#

Required argument
